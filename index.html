Random Dots Generator

Create an image with hundreds of tiny circles (with no overlaps!)

I created this tool as a basis for my data visualization work, but it can also be used for artistic illustrations.

Sometimes, we just want to represent a big number as dots on the screen. In that case, it’s important that all dots are visible. That is why this tool prevents circles from overlapping by default.

But what if we want to highlight some of those dots? Well… you set an amount of circles to be highlighted, with custom colors and sizes.

For instance, if you want to represent that 10% of something. Simply set the amount of dots as 100 and the amount of highlights as 10. You should get results like this:

You can download your creation as a vector (SVG) and further edit it using Figma or Adobe Illustrator. If you prefer, you can also download it as raster (PNG) file.
